# PolyMRL
This is the Python implementation of the methodology described in the MSc. AI Thesis paper "Learning Multimodal Representations of Polymer Molecules for Downstream Prediction Tasks", by Emile Ghedamsi Dhifallah, supervised by Prof. Bernd Ensing. For any questions, feel free to reach out to emile.ghedamsi.dhifallah@student.uva.nl.

## Setup

In order to set up a working Conda environment for model training or finetuning, first clone the repository:

```
git clone https://github.com/<your-username>/PolyMRL.git
cd PolyMRL
```

Then create an environment using the provided .yml file:

```
conda env create -f environment.yml
conda activate PolyMRL
```
